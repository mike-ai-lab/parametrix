1. Overview

PARAMETRIX is a professional SketchUp extension designed for architects, designers, and construction professionals who need to create precise parametric cladding layouts. The extension generates complex multi-face layouts with advanced trimming, rail systems, and pattern synchronization.

Comprehensive layout samples showcasing the extension's versatility beyond basic stone layouts

Multi-Face Layouts

Generate synchronized layouts across multiple selected faces with seamless pattern continuation.

Advanced Trimming

Automatic boolean trimming ensures perfect fit within complex face boundaries.

Rail Systems

Integrated top and bottom rail generation with customizable dimensions and materials.

Pattern Control

Running bond and stack bond patterns with randomization and start position control.

2. Installation & Setup

Installation

Install the PARAMETRIX.rbz file through SketchUp's Extension Manager (Window > Extension Manager > Install Extension). The PARAMETRIX toolbar will appear automatically after installation.

Tip: The extension is also accessible through Extensions > PARAMETRIX Layout Generator menu item.

4. Workflow Guide

4.1 Basic Workflow

Step 1: Select Face(s)

Select one or more faces in SketchUp where you want to apply the cladding layout.

Step 2: Launch Extension

Click the PARAMETRIX toolbar button or use Extensions > PARAMETRIX Layout Generator.

Step 3: Configure Parameters

Set panel dimensions, joints, and pattern preferences. Use Preview Layout to test settings.

Step 4: Generate Final Layout

Click Generate Layout to create the final geometry with automatic trimming.

4.2 Multi-Face Layouts

For complex buildings, select multiple faces and enable "Synchronize Patterns" and "Preserve Corners" for seamless multi-face layouts with proper corner connections.

5. Advanced Features

5.1 Boolean Trimming System

The extension automatically trims generated panels to fit within complex face boundaries, including faces with holes or irregular shapes.

5.2 Rail Systems

Integrated top and bottom rail generation with options for continuous or split rails, customizable dimensions, and separate materials.

5.3 Pattern Synchronization

When multiple faces are selected, the extension can synchronize patterns across faces for seamless architectural continuity.

Professional Tip: The generated components maintain proper 2D behavior and can cut openings in walls and other geometry.
